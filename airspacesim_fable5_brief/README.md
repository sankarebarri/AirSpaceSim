Central idea:

> AirSpaceSim is a reusable air traffic simulation engine driven by environment data and scenario definitions. Learn, Practice, and Simulate are application experiences built on the same core.

The engine is intended to be open sourced and later reused for research, AI, ML, batch simulation, and more sophisticated aviation applications.
